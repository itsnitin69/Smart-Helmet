# 📄 Documentation

This directory contains the supporting documentation and design resources for the **Smart Helmet: IoT-Based Safety & Access System**.

## Contents

### 📌 Circuit connections.png
Illustrates the complete hardware wiring diagram of the Smart Helmet system, including the Arduino Uno, ESP32, MPU6050, MQ-3 Alcohol Sensor, IR Sensor, RF Module, Servo Motor, and other peripheral connections.

### 📱 MIT app block code.png
Contains the MIT App Inventor block programming logic used to establish Bluetooth communication with the ESP32 and control the physical key-inlet access mechanism.

## Purpose

The documentation in this folder helps users understand:

- Hardware architecture
- Circuit connections
- Mobile application logic
- Communication between system components
- Overall project implementation
